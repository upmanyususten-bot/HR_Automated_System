# Resume Screening Application

An automated system for screening resumes using NLP techniques. This application helps recruiters quickly filter and rank candidates based on their resumes.

## Features

- **Resume Parsing**: Extracts text from PDF and Word documents.
- **Skill Extraction**: Identifies key skills and qualifications.
- **Ranking**: Ranks candidates based on job descriptions.
- **Streamlit Interface**: User-friendly web interface for uploading resumes and viewing results.

## Setup Instructions

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/upmanyususten-bot/HR_Automated_System.git
   cd HR_Automated_System
   ```

2. **Create a Virtual Environment**:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows: .venv\Scripts\activate
   ```

3. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the Application**:
   ```bash
   streamlit run resume_screening_app_CORRECTED.py
   ```
